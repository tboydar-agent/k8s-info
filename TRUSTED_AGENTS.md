# Trusted Agents

This repository is maintained by `tboydar-agent` for public Kubernetes / Platform Engineering knowledge tracking.

## Rules

```text
Allowed:
  - Public source summaries
  - Markdown docs
  - ASCII art architecture diagrams
  - Verifiable links
  - Clearly marked assumptions / pending research

Not allowed:
  - Secrets / tokens / credentials
  - Private specific-company internal information
  - Non-public incident details
  - Personal contact information copied from job posts
  - Unverified claims about specific-company internal Kubernetes versions
```

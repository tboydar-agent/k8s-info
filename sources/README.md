# sources/

公開來源與每日追蹤摘要放這裡。

## 使用原則

```text
只保存：
  - 公開 URL
  - 去識別摘要
  - 版本/日期/來源類型
  - 對 platform engineering 的影響判讀

不保存：
  - 私訊
  - 非必要個資
  - recruiter email / phone
  - 內部系統細節
  - 未公開版本資訊
```

## 建議檔名

```text
YYYY-MM-DD.md
```
